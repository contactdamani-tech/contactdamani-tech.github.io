New 2026 VEROLINE® IG2 Sensor Generation

DAMANI Group announces the evolution of its historic VEROLINE® industrial positioning sensor range with the introduction of the new VEROLINE® IG2 generation.

The VEROLINE technology has been widely used for more than two decades for positioning and identification of coke oven machines operating in harsh industrial environments. The new generation aims to modernize the system while maintaining full compatibility with existing installations.

Objectives of the New Generation

The development of the VEROLINE® IG2 generation is based on several key objectives:

  • Ensure long-term sustainability of the VEROLINE solution
  • Maintain compatibility with existing VEROLINE installations
  • Improve overall sensor performance
  • Integrate modern electronic technologies
  • Provide new configuration and diagnostic tools

Improved Hardware Architecture

The new generation introduces an updated electronic architecture integrating improved signal processing and enhanced system reliability.

The system integrates dedicated electronic boards including power supply management, analog processing and RFID identification functions.

The redesigned enclosure includes industrial connectors, LED indicators for system diagnostics and improved mechanical robustness suitable for harsh environments typical of coke oven plants.

Enhanced Sensor Technology

The VEROLINE® IG2 sensor uses a digital architecture based on multiple Hall-effect sensors mounted in differential configuration.

These sensors are designed to compensate automatically for temperature variations, ensuring stable positioning measurements in demanding industrial conditions. :contentReference[oaicite:1]{index=1}

Digital signal processing improves the accuracy of the analog 4-20 mA output used for positioning control while maintaining reliable RFID identification capabilities. :contentReference[oaicite:2]{index=2}

Integrated RFID Identification

The system integrates an RFID antenna directly into the electronic board to improve signal stability and increase reading performance.

This architecture improves repeatability of the identification function and ensures reliable reading of machine identification tags installed along coke oven batteries.

Configuration and Monitoring Tools

The new generation includes dedicated configuration software allowing operators to monitor sensor signals and adjust operating parameters.

The utility software provides graphical monitoring tools enabling:

  • Adjustment of positioning reference parameters
  • Temperature alarm configuration
  • Monitoring of analog positioning signals
  • Monitoring of RFID identification signals
  • Recording of operational data logs

Applications in Coke Oven Plants

VEROLINE® sensors are widely used for positioning and identification of coke oven machines such as:

  • Charging cars
  • Pusher machines
  • Coke guide machines

The VEROLINE® IG2 generation continues to provide reliable positioning and identification functions for these critical machines operating in high-temperature and dusty industrial environments.
